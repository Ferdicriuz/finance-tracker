

# Finance Tracker API

## Description
A backend application for tracking income, expenses, and calculating balances. Built to manage personal financial records efficiently.

## Features
- Add income
- Add expenses
- Calculate balance
- Transaction history
- RESTful API endpoints

## Installation

## bash
git clone https://github.com/Ferdicriuz/finance-tracker.git
cd finance-tracker
npm install


## bash
npm start


## Technologies Used

- Node.js
- Express.js
- MongoDB
- RESTful API
